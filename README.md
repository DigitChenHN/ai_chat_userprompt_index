# AI 对话 Prompt 索引

一个 Tampermonkey 用户脚本，为 AI 大模型聊天界面添加右侧 Prompt 索引导航功能。

## 功能特点

- **自动索引**：自动扫描并提取用户发送的所有 Prompt 消息
- **快速跳转**：点击索引条目快速定位到对应消息位置
- **悬停展开**：鼠标悬停时展开完整索引列表，收起时显示圆点导航
- **可拖动侧边栏**：支持拖动调整侧边栏位置
- **多平台支持**：兼容通义千问、Kimi、Gemini、ChatGPT
- **虚拟滚动支持**：自动加载千问页面的历史消息

## 安装方法

### 前置要求

1. 安装浏览器（Chrome、Edge、Firefox 等）
2. 安装 Tampermonkey 扩展：
   - Chrome/Edge: [Chrome 网上应用店](https://chrome.google.com/webstore/detail/tampermonkey/)
   - Firefox: [Firefox 附加组件](https://addons.mozilla.org/firefox/downloads/latest/tampermonkey/)

### 安装脚本

1. 点击 [ai-prompt-index.user.js](ai-prompt-index.user.js) 文件
2. Tampermonkey 会自动弹出安装页面
3. 点击"安装"按钮

或者：

1. 打开 Tampermonkey 管理面板
2. 点击"创建新脚本"
3. 复制 [ai-prompt-index.user.js](ai-prompt-index.user.js) 的全部内容
4. 粘贴到编辑器中
5. 按 `Ctrl+S` 保存

## 使用方法

1. 打开通义千问、Kimi、Gemini 或 ChatGPT 的聊天页面
2. 发送一些消息后，页面右侧会出现索引侧边栏
3. **悬停**在侧边栏上展开完整索引列表
4. **点击**任意索引条目跳转到对应消息
5. **拖动**侧边栏调整垂直位置

## 技术细节

### 核心功能

- **平台检测**：自动识别当前所在的 AI 平台
- **消息扫描**：使用 CSS 选择器提取用户消息
- **元素定位**：定位器模式解决动态 DOM 问题
- **多策略滚动**：适配不同平台的滚动机制
- **虚拟滚动支持**：自动加载千问页面历史消息
- **Shadow DOM 支持**：穿透 Gemini 样式隔离
- **DOM 变化监听**：实时响应新消息

## 常见问题

### Q: 为什么点击索引后没有反应？

A: 可能原因：
1. 页面结构发生变化，选择器失效
2. 元素被虚拟滚动卸载
3. 平台更新了 class 名称

解决方法：点击侧边栏的"刷新"按钮重新扫描。

### Q: 千问页面为什么只能索引几条消息？

A: 千问使用虚拟滚动技术，只有可见区域的消息会被渲染。脚本会自动滚动加载历史消息，但如果消息数量非常多，可能需要等待一段时间。

### Q: 侧边栏位置不对怎么办？

A: 可以拖动侧边栏调整位置。侧边栏顶部区域支持拖动操作。
